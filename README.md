# MIT Hands-on Deep Learning Notes

## 📌 Description (for GitHub repo)

A structured repository for the MIT Hands-on Deep Learning course, preserving original HTML materials and organizing exercises by section. Each section contains rendered HTML pages linking to the original PDF resources, along with corresponding exercises for practical implementation.

---

## 📖 Overview

This repository is designed to:

* Store and organize MIT Hands-on Deep Learning course materials
* Preserve original HTML files for easy navigation
* Provide direct access to PDF resources via embedded links
* Keep exercises structured and easy to follow

---

## 🗂️ Repository Structure

```
MIT-Hands-on-DL/
│
├── Section-01/
│   ├── section01.html        # Rendered HTML linking to PDF
│   ├── exercises/
│   │   ├── exercise01/
│   │   │   ├── exercise01.html
│   │   │   └── (related files)
│   │   └── exercise02/
│   │       ├── exercise02.html
│
├── Section-02/
│   ├── section02.html
│   ├── exercises/
│   │   ├── exercise01/
│   │   └── exercise02/
│
└── README.md
```

---

## ⚙️ How It Works

* Each `sectionXX.html` file renders content and includes a link to the official PDF of that section
* Exercises are grouped inside an `exercises/` folder within each section
* Each exercise contains its own HTML file and supporting resources

---

## 🚀 Usage

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/MIT-Hands-on-DL.git
   ```

2. Open any section HTML file in your browser:

   ```bash
   open Section-01/section01.html
   ```

3. Navigate to exercises inside each section folder
